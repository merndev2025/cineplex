Cineplex (Final)
================================

A personalized movie recommendation system with user authentication, watch history tracking, and AI-powered recommendations.

Project Structure
---------------
/
├── backend/               # FastAPI backend server
│   └── app/              # Backend application code
├── frontend/             # Frontend web application
│   ├── assets/          
│   │   ├── css/         # Stylesheets
│   │   ├── js/          # JavaScript files
│   │   └── images/      # Static images
│   └── pages/           # HTML pages
└── run.py               # Server runner

Setup Instructions
----------------
1. Requirements:
   - Python 3.8+
   - Modern web browser

2. Quick Start:
   Run the Servers using:
   python run.py


Note: Install the required modules using pip install -r requirements.txt (requirements.txt inside backend) (other modules if errored as needed)
